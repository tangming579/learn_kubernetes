## prometheus安装

1. 下载并解压服务

   ```
   tar xvfz prometheus-*.tar.gz
   cd prometheus-*
   ```

2. 配置prometheus

   https://www.python100.com/html/120606.html

   https://blog.csdn.net/w2009211777/article/details/124005822

3. 启动prometheus

   ```
   ./prometheus --config.file=prometheus.yml
   ```
